    • Hackathon project (backend, ML, DevOps)

      This project was built in asani.io 2 days hackathon. Detects patterns and anomalies in IOT devices data flowing through message broker, using event driven architecture.

      An LSTM based text generation model. Trained on kaggle movie subtitles dataset, 300th dimensional glove embedding layer, single 512 units LSTM layer, a dense layer, and adam optimizer. With ~80% accuracy.

    • NLP Youtube Comments Classifier

      project implements sentiment analysis to determine the overall sentiment (positive, negative, or neutral) of comments on a YouTube video.
